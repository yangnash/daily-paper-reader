## 摘要
组装动作理解是人机协同组装的关键使能技术，但由于细微动作和精细的手物交互，该任务仍具挑战性。我们通过组合上下文微调（CCFT）方法将视觉语言模型（VLM）适配到这一具有挑战性的领域，该方法将组装动作分解为语义元素（动词、物体、工具），并使用模板化问答对微调VLM以识别每个动作元素。这种方法确保了近乎确定性的输出。为了实现有限数据下高效的多任务学习，提出了分层交替训练（LP-AT）方法，通过元素特定的低秩适配器将不同模型层分配给特定动作元素的识别。LP-AT在元素特定适配器之间交替更新权重，减少了跨任务干扰，同时实现了每个适配器的超参数优化。此外，我们从现有组装视频数据集中创建了HA-ViD-VQA和IKEA-ASM-VQA数据集。在这些数据集上的大量实验表明，我们的方法始终优于强动作识别基线，同时提供可解释的元素级预测，支持多种下游应用。

## Abstract
Assembly action understanding is a key enabler for effective human-robot collaborative assembly, yet it remains challenging due to subtle motions and fine-grained hand-object interactions. We adapt vision-language models (VLMs) to this challenging domain with Compositional Context Fine-Tuning (CCFT), a method that decomposes assembly actions into semantic elements (Verb, Object, Tool) and fine-tunes VLMs to recognize each action element using templated question-answering pairs. This approach ensures near-deterministic outputs. To enable efficient and effective multi-task learning under limited data, a Layer-Partitioned Alternating Training (LP-AT) method is presented, which assigns distinct model layers to recognize specific action elements through element-specific low-rank adapters. LP-AT alternates weight updates across element-specific adapters, reducing cross-task interference while enabling per-adapter hyperparameter optimization. Furthermore, we create HA-ViD-VQA and IKEA-ASM-VQA datasets from existing assembly video datasets. Extensive experiments on these datasets demonstrate that our method consistently outperforms strong action recognition baselines while providing interpretable element-level predictions that can support diverse downstream applications.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：人机协作组装（HRC）中，机器人需要理解人类的复杂组装动作，但现有动作识别方法难以应对组装动作的细粒度手物交互、微运动以及语义歧义（如视觉相似但语义相反的动作）。
- **背景**：视觉语言模型（VLM）在通用场景理解上表现出色，但直接用于组装动作理解存在两大障碍：1）VLM生成式输出具有模糊性，不符合HRC对确定性输出的需求；2）缺乏适配组装任务的专门微调方法和结构化问答数据集。
- **整体目标**：提出一种方法，将VLM适配到组装动作理解，实现**近确定性、可解释**的元素级预测，并提升动作识别精度。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- **组合上下文微调（CCFT）**：将组装动作分解为语义元素（Verb, Object, Tool），通过模板化视觉问答对（VQA）微调VLM，每个元素对应一个子任务，输出被限制在有限答案空间，实现近确定性。
- **分层分区交替训练（LP-AT）**：为避免多任务学习中的参数干扰，将VLM的语言编码器层划分为不相交的子组，每个子组由独立的低秩适配器（LoRA）负责一个动作元素。训练时**交替**优化不同元素的适配器，并**合并**到基础模型，在保证参数高效的同时实现跨任务知识迁移。

### 关键技术细节
1. **问题形式化**：给定视频片段x，传统方法直接分类a；CCFT分解为(v, o, t)，训练VLM生成对模板问题的回答。
2. **数据集构建**：将HA-ViD和IKEA-ASM数据集转换为VQA格式，为每个元素生成模板问题-答案对（如“What is the action verb?” → “screw”）。
3. **模型与训练**：使用Qwen2.5-VL-7B，仅微调语言编码器（28层），冻结视觉编码器。LP-AT中，HA-ViD-VQA：层0-9负责动词，10-23负责对象，24-27负责工具；IKEA-ASM-VQA：层0-13动词，14-27对象。每个适配器独立配置秩（rank=256）、学习率、缩放因子。
4. **训练流程**：每个epoch内，依次训练每个子任务的LoRA适配器，更新后合并到基础模型，再训练下一个任务，实现交替优化。
5. **评估协议**：使用基于MiniLM嵌入的余弦相似度（阈值0.95）比较预测字符串与真实标签，计算元素级和整体动作准确率。
